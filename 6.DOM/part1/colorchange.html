This is the Header

You will use the javascript file to continually change the size of the header, but first we need to learn how to use javascript to actually grab the HTML objects form the DOM

Here are some important document attributes:

  • document.URL-- This is the actual URL of the website
  • document.body -- This is everything inside the body
  • document.head -- This is everything in the head of the page
  • document.links -- These are all the links on the page

Then we also have methods we can use to grab HTML elements:

  • document.getElementById()-- Retruns the element with the id
  • document.getElementByClassName()-- Returns list of all elements belonging to a class
  • document.getElementByTagName()-- Returns list of all element with the tag
  • document.querySelector()-- Returns the first object matching the css style selector
  • document.querySelectorAll()-- Returns all objects matchin the css style selector

The first of these arae pretty self explanatory. The querySelector() can accept CSS style selector, so you can use # or if you want ids or classes. Basically a more generalized method where you can type any css style. The different between querySelector() and querySelectorAll() will return a list of all matches, querySelector() just return the first match
